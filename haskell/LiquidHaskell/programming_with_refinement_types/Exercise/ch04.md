# Polymorphism

## Exercise 4.1 (Vector Head)

```haskell
{-@ head' :: Vector a -> Maybe a @-}
head' :: Vector a -> Maybe a
head' vec
  | null vec = Nothing
  | otherwise = Just $ V.head vec
```

### 別解


```haskell
{-@ head'' :: Vector a -> Maybe a @-}
head'' :: Vector a -> Maybe a
head'' vec | length vec == 0 = Nothing
           | otherwise = Just (vec ! 0)
```
