# 動作環境

```hs
{-# LANGUAGE TypeOperators #-}
{-# LANGUAGE DataKinds #-}
{-# LANGUAGE OverloadedLabels  #-}
{-# LANGUAGE OverloadedStrings #-}
{-# LANGUAGE TypeApplications  #-}

import Data.Extensible
import Control.Lens         ((^.))

```

## hpack

```yaml
dependencies:
- extensible
- lens

default-extensions:
- TypeOperators
- DataKinds
- OverloadedLabels
- OverloadedStrings
- TypeApplications
```

## cabal

```
build-depends:
      extensible
    , lens

default-extensions: TypeOperators DataKinds OverloadedLabels OverloadedStrings TypeApplications
```
